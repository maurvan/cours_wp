== Description ==

Custom Font Uploader is a powerful, extendable, google webfonts plugin that helps you to use google fonts in your theme .
It also allows you to upload your own custom font to your site and use them using custom css.
Allow to upload any number of fonts, it does not have limit and you also do not need any API to use it. All fonts are saved at your server. We do not serve any files from our server.

== Manual Installation ==
You can download and install the plugin using built in WordPress plugin installer.

The manual installation method involves downloading the plugin  and uploading it to your webserver via your FTP application.

1. Download the plugin file to your computer and unzip it.
2. Upload the entire "custom-font-uploader" folder to the /wp-content/plugins/ directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.

= Will Custom Font plugin With my theme ? =
